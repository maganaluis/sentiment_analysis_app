STAT-542 Project - Sentiment Analysis

SVM Model with TFID Vectorizer.

Our objective for this project was to create binary classifier that could classify word vectors into negative or positive based on training data from IMDB movie reviews. My approach as suggested by Prof. Liang, was to create a 40K vocabulary using the documents from the test dataset, without stemming or normalization and up to 2-ngrams.

Then reduce the vocabulary using the two-sample t-statistic shown below, and otbain the top 3K words using the absolute value of the p-val generated. This results in a more meaningful vocabulary, and easer to explain for the model.

My final model for this project was a Linear SVC, since text data is often linearly separable and it was easier to train compared to Neural Networks or Gradient Boosted Trees.

Given: $$ X_1, X_2, ... X_m \space Y_1, Y_2, ... Y_n $$ $$ \begin{equation} p=\frac{\bar{X} - \bar{Y}}{\sqrt{\frac{var(X)}{m} + \frac{var(Y)}{n}}}\ \end{equation} $$
